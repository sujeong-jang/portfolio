길찾기 App

안드로이드 프로그래밍 수업에서 팀프로젝트로 개발한 길찾기 App입니다. 친구를 약속장소로 만나러 갈 때, 처음 가본 길이나 복잡한 길에서 지도를 보면서도 올바른 길을 찾지 못하고 헤매는 상황이 종종 있습니다. 이와 같은 상황을 해결하기 위해서 사용자의 기준에서 로드뷰를 자동으로 보여주며 길 안내를 해주어 길을 원할하게 찾을 수 있는 App을 개발하였습니다. 또한 로드뷰 하단의 미니맵으로 검색과 주변의 음식점이나, 카페 등 300M 반경의 주변정보도 알 수 있으며, 자신의 현재위치를 친구와 공유할 수 있습니다.


- 개발 인원 : 2명
- 개발 기간 : 2018.10-2018.12
- 담당 분야 : Google Map Api, Place Api 사용하여 미니맵(검색, 마크 기능) 구현, Firebase 사용하여 로그인, 회원가입 구현
- 개발 언어 : Java
- 사용 기술 : Google Map Api, Place Api, Directions Api, Firebase(Realtime database, Authentication)

1인 가구를 위한 반려견 IoT 서비스

최근 반려견을 키우는 1인 가구가 증가하는 추세입니다. 하지만 1인 가구의 특성상 집을 비우는 시간이 많기 때문에 집 밖에서도 반려견을 케어할 수 있는 IoT서비스를 개발하였습니다. 주인은 라즈베리와 연동된 카메라를 통해 반려견의 상태를 실시간으로 체크하고 목소리를 들려줄 수 있으며, 시간을 맞춰 사료를 배식하고 반려견에게 안정감을 주는 노래를 들려줄 수 있습니다. 또한, 추위보다 더위를 많이 타는 반려견을 위해 일정 불쾌지수 도달 시 자동으로 선풍기를 작동시키며, 딥러닝을 사용해 반려견 상태를 분석하여 직접 카메라로 확인하지 않아도 화면을 통해 반려견의 현재 상태를 간단히 확인할 수 있습니다.


- 개발 인원 : 4명
- 개발 기간 : 2019.03-2019.06
- 담당 분야 : MQTT 프로토콜을 이용한 라즈베리파이와 안드로이드 기기 간의 데이터 통신, Firebase 로그인/회원가입 기능, 딥러닝(이미지 분류)
- 개발 언어 : Java, Python
- 사용 기술 : Android, Deep learning(Keras), OpenCV, MQTT, Firebase
- 수상 내역 : 교내 공학 경진대회(2019.09.27) - 은상, 광운대 해커톤 대회(2019.08.23) - 장려상,
캡스톤 디자인(2019.06.07) - 혁신상
